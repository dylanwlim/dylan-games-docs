# How It Works

Dylan Games is documented here from the public user's point of view.

## Hub shell

The homepage opens directly into the game launcher with Snake selected.

## Playable starter

Snake is the current playable browser game.

## Planned slots

The launcher shows future games as planned entries instead of playable games.

## Publishing Model

The public guide files live under `docs-public/` in the source repository. The publishing workflow captures the current homepage screenshot, validates the public docs allowlist, and mirrors the result into `dylanwlim/dylan-games-docs`.
